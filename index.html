Simplexity

Simplexity is an application made using R and rshiny package. It has an intuitive UI that offers solvers for Polynomial Regression and Quadratic Spline Interpolation. It also presents the Diet Problem and its corresponding solvers depending on user input.

Slam Go!

SLAM GO! is a mobile slambook app that allows the user to add their friends through a slambook format, see their friends list, create an account that stores their credentials and slambook data.
